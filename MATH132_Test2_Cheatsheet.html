§6.7 Pumping Water (Conical Tank)

W = ∫ ρ·g · A(y) · (lift distance) dy
🔑 SETUP is the whole challenge. Use similar triangles to find r in terms of y.

Cone: H=6m, top radius R=1.5m (inverted)

Similar triangles:r/y = 1.5/6 = 1/4 → r = y/4
Cross-section area:A(y) = π(y/4)² = πy²/16
Lift distance:(6-y) for slice at height y
W = (1000·g·π/16) ∫[0,6] y²(6-y) dy ≈ 207,832 J
⚠ Half-depth ≠ half work! Lower water is lighter AND lifted less → only ~2% of full work.

General steps:

1. Sketch tank, label y from bottom

2. Similar triangles → r as function of y

3. A(y) = πr²

4. Lift = (height of top) - y

5. W = ∫ρ·g·A(y)·lift dy (ρ_water = 1000 kg/m³, g = 9.8)

§7.1-7.2 Exponential Models

P(t) = P₀·eᵏᵗ   OR   P(t) = P₀·(rate)ᵗ
ModelFind t
Continuous: P=P₀eᵏᵗt = ln(P/P₀)/k
% rate r/yr: P=P₀(1+r)ᵗt = ln(P/P₀)/ln(1+r)

§7.3 Hyperbolic Functions — START HERE if lost

🆘 Think of these like sin/cos but with e instead of circles.
DefinitionFormula
cosh x= (eˣ + e⁻ˣ)/2
sinh x= (eˣ - e⁻ˣ)/2
tanh x= sinh x / cosh x = (e²ˣ-1)/(e²ˣ+1)
sech x= 1/cosh x

Identities:

cosh²x - sinh²x = 1    1-tanh²x = sech²x
cosh 2x = cosh²x + sinh²x

Derivatives (MEMORIZE these):

d/dx[sinh x]= cosh xd/dx[cosh x]= sinh x
d/dx[tanh x]= sech²xd/dx[sech x]= -sech x·tanh x

Integrals:

∫ cosh 2x dx= ½sinh 2x + C [u=2x]
∫ sinh x/(1+cosh x) dx= ln(1+cosh x)+C [u=1+cosh x]
∫ tanh²x dx= x - tanh x + C [use: tanh²=1-sech²]

Proving Identities — always expand using definitions:

To prove anything: replace cosh/sinh with (eˣ±e⁻ˣ)/2, simplify algebra, done.

Ex: tanh x = sinh/cosh = [(eˣ-e⁻ˣ)/2]/[(eˣ+e⁻ˣ)/2] → multiply top&bottom by eˣ → (e²ˣ-1)/(e²ˣ+1) ✓
